HTML

DEFINITION - HTML (Hypertext Markup Language) is a coding language that is used to make and structure any web page and the content in it.

HTML ELEMENTS - The content of the HTML document is contained inside HTML elements. HTML element is a fundamental unit to create a webpage.

HTML TAG - HTML tag is a word that starts with ‘<’ and ends with ‘>’ and is used to indicate the starting and end of an HTML element.

Some of the HTML tags are-

<!DOCTYP>

<html>

<body>

<p>

<h1>

<br>

<hr>

Many HTML elements have an opening and closing tag — for example, a h1 (heading) element has a <h1> tag, followed by the heading text, and then tag at the end of text.

For Example - <h1>The World<h1>

Some elements like <br>, <img> and <hr> don’t have any closing tag.

For Example - <img src=”mango.jpeg” alt-”a mango”>

HTML ATTRIBUTES - HTML attributes can be defined as alterers used to describe the properties of HTML elements.

Some of the HTML attributes are -

href - describes the URL (web address) for a link

alt - describes an alternative text for an image

Id - names a unique id of an element

src - states the URL of an image

style - identifies an inline CSS style for an element

INLINE ELEMENTS - Inline elements are displayed in a line , they do not start on a new line.Inline elements occupy the space as needed within the space of the main element.

Some of the inline elements are - <a>,<span>,<img>,<button>,<input> etc.

BLOCK ELEMENTS - Block elements start on a new line and may contain inline elements.Unlike inline elements,block elements occupy the whole space of the container.

Some of the block elements are <p>, <h1>, <h2>, <h3>, <h4>, <h5>, <h6>,<ul>,<ol>,<p>.

CSS

DEFINITION - Cascading Style Sheets(CSS) is a stylesheet language for styling the content of a webpage written in HTML.

CSS RULE - CSS rule is basically a statement for styling one or more elements in HTML.The rule follows a specific structure.

It consists of a selector and declaration that is surrounded by curly brackets:

selector{ declaration }

Selector - HTML tags such as <body>,<p>,<li>,<ul> and <h1> are called selectors when they are stated in a CSS rule.

Declaration - The declaration contains the characteristic of the selector and the value of the characteristic.The structure is written like:

selector{characteristic:value;}

For Example:

                
                    p 
                    {
                        background-color: green;
                    }

Where

p is the selector

background-color is the characteristic of the selector

green is the value of the characteristic

background-color:green;is the declaration
